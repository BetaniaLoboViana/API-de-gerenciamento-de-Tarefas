# 📋 Tarefas API

Uma API REST simples para gerenciar tarefas, construída com Spring Boot.

## 🚀 Funcionalidades

- Criar uma nova tarefa
- Listar todas as tarefas
- Atualizar tarefas
- Marcar como concluída
- Deletar tarefas

## 🛠️ Tecnologias

- Java 17+
- Spring Boot
- Spring Data JPA
- H2 Database (para testes)
- PostGreSQL
- Swagger/OpenAPI
